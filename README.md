# Project Overview

This project simulates a real-world data engineering workflow for a mental health platform similar to LePal.ai. 

It focuses on building scalable data pipelines to ingest, process, and analyze user engagement data from therapy sessions and AI-driven interactions.

The pipeline enables tracking of user activity, session behavior, and feature engagement to support data-driven product decisions.
